Banco Guará

Este projeto consiste no desenvolvimento de um sistema de Internet Banking voltado para operações bancárias simples, como depósito, saque e transferência entre usuários.

Um diferencial desse projeto foi o desenvolvimento das configurações de rendimento e de cheque especiais, que deveriam ser configuradas globalmente no momento da execução e precisavam afetar todas as contas.

Minhas contribuições para este projeto foram relacionadas a definição do MER do Banco de Dados e criação dos scripts SQL do mesmo.

Tecnologias

Python, Flask, MySQL, CSS, HTML e Javascript.
